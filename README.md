# shinyreturns
Online return calculator

This is a Shiny App providing an online return calculator using the package [simulreturns](https://github.com/Layalchristine24/simulreturns).

First of all, load the Github library [simulreturns](https://github.com/Layalchristine24/simulreturns) by running: 

devtools::install_github("https://github.com/Layalchristine24/simulreturns")
